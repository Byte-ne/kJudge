# kJudge: Minimalist Local Judge CLI

**kJudge** is a high-performance, minimalist command-line tool designed for competitive programmers. It manages your local workspace, fetches test cases from online judges, and runs your solutions with precise time/memory monitoring.

## Why kJudge?

- **Zero-Friction Context**: `init`, `add`, `run`. No complex IDE setup required.
- **Fast Scaffolding**: Create entire contest structures in seconds with `kjudge contest`.
- **Hybrid Fetching**: Robust sample retrieval using direct scraping with VJudge mirroring fallback.
- **Modern UI**: Clean, color-coded terminal reports with ASCII safety for Windows compatibility.

## Get Started in 3 Steps

1.  **Init**: `kjudge init --lang cpp`
2.  **Fetch**: `kjudge fetch cf:4A`
3.  **Run**: `kjudge run main.cpp`
